    Quick Command Reference

    # Development
bun install              # Install dependencies
bun dev                  # Start dev server
bun build                # Build for production

# Testing
bun test                 # Run tests in watch mode
bun test:ui              # Open test UI
bun test:coverage        # Generate coverage report

# Code Quality
bun lint                 # Run ESLint

    Key Features

    • 🎨 Rich Text Editor - Powered by Plate.js with extensive plugins
    • 🔐 Authentication - Google OAuth integration
    • 🌍 Internationalization - Multi-language support (EN, FR)
    • 📱 Responsive Design - Mobile-first approach with Tailwind CSS
    • 🧪 Comprehensive Testing - Unit and component tests with MSW mocking
    • 📊 Graph Visualization - Interactive graph editor and mapping tools
    • 🎯 Admin Dashboard - Administrative interface for content management

    Getting Help

    1. Search the docs - Most questions are answered in the documentation
    2. Check troubleshooting - See TROUBLESHOOTING.md
    3. Review test files - Tests serve as living documentation
    4. Ask questions - Create an issue or reach out to the team
